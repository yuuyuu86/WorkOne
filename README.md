# WorkOne

連絡・学校・予定系のサービスを、ひとつのアプリにまとめて使える macOS 向けの
連絡専用アプリです。Gmail・Slack・Google Classroom・Google Calendar などを
行き来しながら、今日やること・未読・予定をホーム画面でまとめて確認できます。

ログインは各サービスの公式ログイン画面で行います。WorkOne 側で ID・パスワードを
入力したり保存したりすることは一切ありません。

---

## インストール

1. [Releases](https://github.com/yuuyuu86/WorkOne/releases) から最新の
   `WorkOne-x.y.z-arm64.dmg` をダウンロードします（Apple Silicon / M1 以降向け）。
2. dmg を開き、**WorkOne**を「アプリケーション」フォルダにドラッグします。
3. 初回起動時に「開発元を確認できません」と表示された場合は、
   **アプリを右クリック →「開く」→「開く」** を選んでください（初回のみ）。
   - それでも開けないときは、`システム設定 → プライバシーとセキュリティ` の下部に出る
     「このまま開く」を押してください。

> このアプリは現在 Apple の署名を付けていないため、初回のみ上記の操作が必要です。

## できること

### ホーム
- 今日の予定（カレンダー）、Classroom の課題、未読サマリー、通知をひと目で確認
- ホームの各ウィジェットを **ドラッグで並び替え／表示・非表示／横並び** に自由にカスタマイズ
  （右上の「編集」ボタンから）
- 天気や時間帯に応じたあいさつを表示

### サービス
- Gmail / Slack / Google Classroom / Google Calendar などを切り替えながら利用
- **Slack は複数ワークスペース対応**。サインイン済みのワークスペースを自動で見つけて
  ワンクリックで追加できます
- サイドバーはカテゴリ分け・ドラッグ並び替え・幅調整・自動非表示に対応
- 各サービスの未読件数をバッジ表示

### 予定・課題
- 今日の予定を「終日」「時刻つき」に整理して表示
- Classroom の課題を「今日／今週／次週／それ以降／期限なし」で分類
- 締切が近い課題を通知

### 便利機能
- **コマンドパレット（⌘K）** … 履歴・課題・予定をまとめて検索してすばやく移動
- **集中モード**／**おやすみ時間（DND）**／通知音の ON/OFF
- ダークモード
- 設定のエクスポート／インポート

## 通知について

- 各サービス側で「デスクトップ通知」をオンにすると、サービス自身が OS 通知を出します。
- 受け取った通知は **Inbox 画面に集約**されます。
- ウィンドウを閉じてもアプリはバックグラウンドに常駐し、通知を受け続けます。
  完全に終了するにはメニューの「終了」を使ってください。
- 設定の「起動時に全サービスを自動で読み込む」をオンにすると、開いていない
  サービスの通知・未読も受け取れます。

## プライバシー

- **ID・パスワードは保存しません。** ログインは各サービスの公式画面で行います。
- **メール本文・チャット内容・添付ファイル・ログイン情報は保存しません。**
- 今日の予定や Classroom の課題は、あなたがログイン中のアカウントから
  **予定名・課題名・日時などの表示用の情報だけ**を読み取り、**お使いの端末内にのみ**
  キャッシュしてホームに表示します（外部送信はしません）。
- アプリの設定（追加したサービス、並び順、ホームのレイアウトなど）は、
  端末内のローカル領域にのみ保存されます。

データを消したいときは `設定 → データを削除` から、保存内容やログインセッションを
まとめて削除できます。

## アップデート

新しいバージョンが公開されると、起動時にアプリ上部へお知らせのバナーが表示されます。
「ダウンロード」を押すと最新版のページが開くので、新しい dmg をダウンロードして
上書きインストールしてください（勝手に自動更新することはありません）。

## よくある質問（FAQ）

**Q. Windows / Intel Mac では使えますか？**
A. 現在は Apple Silicon（M1 以降）の macOS のみ配布しています。Windows 版・Intel Mac 版は今のところ予定していません。

**Q. 「開発元を確認できないため開けません」と出て起動できません。**
A. Apple の署名（ノータライズ）を付けていないための表示です。アプリを右クリック →「開く」→「開く」で初回のみ回避できます。それでも開けない場合は `システム設定 → プライバシーとセキュリティ` の下部に出る「このまま開く」を押してください。

**Q. ID・パスワードを入力する画面が WorkOne 内にありません。安全ですか？**
A. WorkOne は各サービスの公式ログイン画面をそのまま使います（IDやパスワードを WorkOne が受け取る・保存することはありません）。詳しくは [プライバシー](#プライバシー) をご覧ください。

**Q. メールの本文やチャットの内容は WorkOne に保存されますか？**
A. 保存しません。カレンダーの予定名や Classroom の課題名など、ホーム画面に表示するための最小限の情報のみを端末内にキャッシュします。外部への送信もありません。

**Q. Gmail の未読件数やカレンダーの予定が表示されません。**
A. 一度サイドバーからそのサービスを開くと情報を取得できます（未読・予定はアプリが表示中/直近に開いたサービスから取得する仕組みのため）。しばらく操作していないサービスは表示が古いままのことがあります。

**Q. 通知が来ません。**
A. 各サービス側の「デスクトップ通知」設定がオンになっているか確認してください。また、設定の「起動時に全サービスを自動で読み込む」をオンにすると、開いていないサービスの通知も受け取りやすくなります。

**Q. アプリを閉じても常駐しているのはなぜですか？**
A. バックグラウンドで通知を受け取り続けるためです。完全に終了したい場合はメニューの「終了」を選んでください。

**Q. 新しいバージョンが出たらどうすれば良いですか？**
A. 自動更新はありません。起動時にお知らせバナーが出たら「ダウンロード」から最新の dmg を取得し、上書きインストールしてください。

**Q. LINE や Notion、AI ツールには対応しますか？**
A. 対応の予定はありません。連絡・学校・予定に専念するため、LINE / Notion / Canva / Figma / GitHub / AI ツール系には対応しません。パスワードの保存、本文の保存、自動ログインも行いません。

**Q. サービスの並び順やホームのレイアウトを元に戻したいです。**
A. ホーム編集モードの「リセット」ボタンでウィジェットのレイアウトを初期状態に戻せます。サービス一覧の並びはサイドバーの編集からドラッグで調整してください。

**Q. データを完全に消したいです。**
A. `設定 → データを削除` から、保存されているキャッシュやログインセッションをまとめて削除できます。
